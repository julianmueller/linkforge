# Citing LinkForge

If LinkForge has been useful for your research or project, please consider citing it. This helps us track the impact of the tool and justifies further development.

## Preferred Citation

### BibTeX
```bibtex
@software{linkforge_2026,
  author       = {Patouossa Mounchili, Arouna},
  title        = {{LinkForge: The Linter & Bridge for Robotics}},
  year         = {2026},
  version      = {1.2.3},
  publisher    = {GitHub},
  journal      = {GitHub repository},
  howpublished = {\url{https://github.com/arounamounchili/linkforge}}
}
```

### APA
Patouossa Mounchili, A. (2026). LinkForge: The Linter & Bridge for Robotics (Version 1.2.3) [Computer software]. https://github.com/arounamounchili/linkforge

## GitHub Citation
You can also use the **"Cite this repository"** button in the [LinkForge GitHub sidebar](https://github.com/arounamounchili/linkforge) to export the citation in various formats.
